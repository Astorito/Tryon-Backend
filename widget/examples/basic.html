Integration Code

To add this widget to your website, simply include this line in your HTML:

<script src="https://cdn.tryon.app/widget.js"></script>

The widget will automatically:

  • Create a floating button in the bottom-right corner
  • Inject all necessary styles
  • Handle file uploads and image processing
  • Manage user interface and interactions

Configuration (Optional)

You can configure the widget with custom API settings:

<script>
  window.TRYON_CONFIG = {
    apiUrl: 'https://your-backend.vercel.app',
    apiKey: 'your-api-key'
  };
</script>
<script src="https://cdn.tryon.app/widget.js"></script>

üõí Platform-Specific Integration

Shopify

  1. Go to Online Store ‚Üí Themes
  2. Click Edit code
  3. Find theme.liquid file
  4. Add before closing </body> tag:
<script src="https://cdn.tryon.app/widget.js" data-tryon-key="sk_live_xxx"> </script>

WooCommerce / WordPress

  1. Go to Appearance ‚Üí Theme File Editor
  2. Edit footer.php
  3. Add before closing </body>:
<?php if (get_option('tryon_api_key')) { ?> <script src="https://cdn.tryon.app/widget.js" data-tryon-key="<?php echo get_option('tryon_api_key'); ?>"> </script> <?php } ?>

React / Next.js

import { useEffect } from 'react'; export default function App() { useEffect(() => { const script = document.createElement('script'); script.src = 'https://cdn.tryon.app/widget.js'; script.setAttribute('data-tryon-key', process.env.REACT_APP_TRYON_API_KEY); script.async = true; document.body.appendChild(script); }, []); return <div>Your App</div>; }

Vue.js

export default { mounted() { const script = document.createElement('script'); script.src = 'https://cdn.tryon.app/widget.js'; script.setAttribute('data-tryon-key', process.env.VUE_APP_TRYON_API_KEY); script.async = true; document.body.appendChild(script); } }

üîë How to Get Your API Key

  3. Go to Settings ‚Üí Integration
  4. Copy your Live API Key (starts with sk_live_)
  5. Replace YOUR_API_KEY_HERE with your actual key

üîß Troubleshooting

Widget doesn't appear?
  1. Open DevTools (F12) and check Console
  2. Look for errors starting with [Tryon Widget]
  3. Verify data-tryon-key attribute is present
  4. Check Network tab - widget.js should load successfully
Images won't generate?
  1. Verify API key is valid (not expired or revoked)
  2. Check backend URL is reachable (if custom)
  3. Ensure file sizes are under 10MB
  4. Check browser console for error messages
Reset widget state?

Run this in browser console to clear stored data:

Object.keys(localStorage) .filter(k => k.startsWith('tryon-widget-')) .forEach(k => localStorage.removeItem(k));

üîê Security Best Practices

  1. Use HTTPS Only - Widget requires secure context in production
  2. Protect API Keys - Never hardcode in HTML, use environment variables
  3. Rotate Keys Regularly - Change compromised keys immediately
  4. Enable CORS - Backend handles CORS automatically
  5. CSP Headers - Allow widget domain: script-src 'self' https://cdn.tryon.app

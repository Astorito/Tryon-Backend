üêõ Troubleshooting

Widget doesn't appear

Check your browser console for error messages. Common issues:

  • Missing or incorrect data-tryon-key attribute
  • API key not authorized in the Metrics service
  • CORS issues (ensure backend allows your domain)

Image generation fails

Verify:

  • API key is valid and has daily generation quota remaining
  • User photos are in supported formats (JPG, PNG, WebP)
  • Backend is online: /health
üí° Tip: Open your browser console (F12) to see detailed error messages and debug information.
